Privacy Policy

Last updated: November 26, 2025

BladePeek is a mobile application designed to assist wind turbine technicians in capturing and organizing internal blade inspection photos and associated metadata. This privacy policy explains how the app handles user data and ensures compliance with Apple App Store guidelines, including Section 5.1 related to user privacy and data use.

1. Data Collection

  • Photos: The app captures inspection photos using the device camera. These images remain stored locally on the device.
  • Metadata: BladePeek stores basic inspection metadata related to the damage (e.g. blade section, chamber, observed damage type, radius). This information is entered manually by the user.
  • Inspection date: The app stores the date of inspection (format YYYY-MM-DD).
  • License token / device access code: A device-specific access code is entered by the user to generate a local license token used to unlock app functionality. The original access code is not stored, transmitted, or linked to personal identifiers. The license token is stored only on the device and is never used for tracking or analytics.
  • Personal information: The app does not collect or request names, email addresses, technician IDs, or any other personal identifiers.
  • Location data: GPS or precise location data is not collected or used.
  • Accounts: BladePeek does not create or manage user accounts, and does not offer account registration or sign-in.
  • Analytics or crash reporting: The app does not collect analytics, usage statistics, or crash reports.

2. Data Storage

  • All data (photos and metadata) is stored locally on the user's device within the app’s sandboxed storage.
  • BladePeek does not upload or sync data to any external cloud service or server controlled by the developer or third parties.
  • The user may choose to manually export or share data using standard iOS features (e.g., the Files app, zip sharing, AirDrop, email, etc.).

3. Data Sharing

BladePeek does not share data with any third parties. Specifically:

  • Data is not transmitted to Apple, Google, Expo, or any other third-party service providers by the app.
  • There is no backend server collecting or storing data.
  • The app does not use advertising, analytics, or crash-reporting SDKs.
  • The app does not send any personal data or inspection data to third-party AI services or external machine learning APIs.

4. User Control and Permissions

  • Users can delete photos and metadata from the app at any time using the app’s built-in tools or through the iOS Files app, depending on how data has been exported.
  • Users can export their data in the form of folders, CSV logs, and image files using standard iOS sharing options.
  • No personal information is required to use the app.
  • Camera access is requested only when capturing photos, and solely for the purpose of taking inspection images. Users can revoke this permission at any time via the iOS Settings app.

5. Security and Offline Use

  • BladePeek is designed for secure, offline use. No data leaves the device unless the user explicitly chooses to export it.
  • There are no background network processes or automatic data transmissions.
  • BladePeek does not track users across apps or websites and does not use any advertising identifiers (such as IDFA).

6. Children’s Privacy

BladePeek is intended for use by professional wind turbine technicians and is not directed to children. The app does not knowingly collect any personal information from children under the age of 13.
