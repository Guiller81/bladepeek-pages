Privacy Policy

Last updated: October 6, 2025

BladePeek is a mobile application designed to assist wind turbine technicians in capturing and organizing internal blade inspection photos and associated metadata. This privacy policy explains how the app handles user data and ensures compliance with Apple App Store guidelines.

1. Data Collection

  • Photos: The app captures inspection photos using the device camera. These images remain stored locally on the device.
  • Metadata: BladePeek stores basic inspection metadata related to the damage (e.g. blade section, chamber, observed damage type, radius). This information is entered manually by the user.
  • Inspection date: The app stores the date of inspection (format YYYY-MM-DD).
  • Device ID: The device ID is manually entered by the user to generate a license token for app activation. The device ID is not stored or transmitted.
  • Personal information: The app does not collect or request names, email addresses, technician IDs, or any other personal identifiers.
  • Location data: GPS data is not collected or used.
  • Analytics or crash reporting: The app does not collect any analytics or usage data.

2. Data Storage

  • All data (photos and metadata) is stored locally within the app folder on the user's device.
  • BladePeek does not upload or sync data to any external cloud service or server.
  • The user may choose to manually export or share data using standard iOS features (e.g., the Files app, zip sharing, AirDrop, etc.).

3. Data Sharing

BladePeek does not share data with any third parties. Specifically:

  • Data is not transmitted to Apple, Google, Expo, or any third-party service providers.
  • There is no backend server collecting or storing data.
  • The app does not use crash reporting, analytics, or advertising SDKs.
  • No third-party libraries are used that collect user data.

4. User Control and Permissions

  • Users can delete photos and metadata from the app at any time using the app’s built-in tools or through the iOS Files app.
  • Users can export their data in the form of folders, CSV logs, and image files.
  • No personal information is required to use the app.
  • Camera access is requested only when capturing photos. Users can revoke this permission at any time via iOS Settings.

5. Security and Offline Use

  • BladePeek is designed for secure, offline use. No data leaves the device unless the user chooses to export it.
  • There are no background processes or data transmissions.
