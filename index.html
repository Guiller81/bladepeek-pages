BladePeek

BladePeek is a field inspection app designed to streamline internal wind turbine blade inspections. It helps technicians document, manage, and report blade conditions efficiently, following site-specific requirements and capturing essential damage data.

App Workflow Overview

  1. Landing Screen & Tabs: The main screen displays the three app tabs: Inspect Turbine, Manage Inspections, and License.
    Screenshot: Main Screen
  2. License Activation: The user must visit the License tab and click Get Device Access Code to retrieve a unique device ID. This code is sent to the engineering team for license generation.
    Screenshot: License Tab
  3. Token Entry: Once the license token is received, the user enters it and taps Verify License to activate the app.
  4. Starting an Inspection: In the Inspect Turbine tab, the user selects a wind farm and WTG ID. If the site has specific requirements, they are shown in a popup before inspection begins.
    Screenshot: Site Requirements
  5. Camera View: The camera interface is used to take inspection photos. The UI includes:
    • Top Left: Site and WTG ID
    • Right Side (Top to Bottom):
      • Shutter button
      • Flash toggle
      • Blade selector
      • Chamber selector
      • Radius selector
      • Damage (opens form)
      • Done (returns to home)
    Screenshot: Camera View
  6. Damage Form: Complete required fields such as component, subcomponent, and measurements, then save.
    Screenshot: Damage Form
  7. Manage Inspections: In this tab, the user sees a list of all inspections performed. Each inspection can be deleted or exported.
    Screenshot: Manage Inspections
  8. Manage Damages: Inside a selected inspection, the user can click Manage Damages to edit or delete previous damage reports.
    Screenshot: Manage Damage Table
  9. Review Pictures: Users can browse pictures taken and also create new damage entries from them.
    Screenshot: Inspection Picture View

Note: License tokens are managed internally. Please contact your engineering department with the device code to request activation. Personal contact information is intentionally not listed here.
