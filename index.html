BladePeek

An offline-first mobile app for internal blade inspections.

What is BladePeek?

BladePeek is a lightweight inspection tool designed for wind turbine technicians. It allows users to capture internal blade damage photos, record critical metadata, and export the results in structured folders and CSV logs — all without requiring internet access.

Key Features

  • Capture inspection photos directly from the in-app camera
  • Tag each photo with chamber (LE, TE), blade face (A/B), and radius markers
  • Log damage dimensions (length, width, depth) and include freeform comments
  • Organize inspections per site, turbine, and blade
  • Export image folders and CSV damage logs via iOS Files or AirDrop
  • Simple license system using Device Access Code (no login required)

App Screenshots

Offline and Secure

BladePeek is fully functional offline. No data is uploaded or synced to the cloud. All information stays on the user’s device unless they choose to export it manually.

Designed for Technicians

The interface is optimized for field use: large buttons, fast photo capture, and minimal navigation. BladePeek helps standardize damage reporting across teams and reduce manual data entry.
